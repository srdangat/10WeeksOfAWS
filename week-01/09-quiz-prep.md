## Quick Revision

- Region: geographic AWS location.
- Availability Zone: isolated data center location inside a Region.
- Edge Location: closer location for content delivery.
- IAM User: long-term identity.
- IAM Role: temporary access.
- IAM Policy: JSON permissions.
- Least privilege: only required access.
- STS: temporary credentials.
- OIDC: identity federation without storing AWS access keys.
